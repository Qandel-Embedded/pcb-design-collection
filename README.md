# PCB Design Collection

A collection of custom PCB designs for various embedded systems projects, ranging from simple breakout boards to complex multi-layer industrial designs.

## Designs Included

### Motor Controller Board
- 4-layer PCB, STM32 + DRV8305 gate drivers
- CAN Bus interface, 48V/30A capability

### IoT Sensor Hub
- 2-layer, ESP32 + multiple sensor interfaces
- Battery management with solar charging

### Robot Main Controller
- 6-layer high-speed design
- Raspberry Pi CM4 + STM32 co-processor

## Tools
- **Primary:** Altium Designer 22
- **Secondary:** KiCad 7
- **Simulation:** LTSpice, HyperLynx

## Standards
- IPC-2221 design guidelines
- EMC compliance (CE/FCC pre-compliance)
